## Install Comark

::code-group

```bash [pnpm]
pnpm add comark
```

```bash [npm]
npm install comark
```

```bash [yarn]
yarn add comark
```

```bash [bun]
bun add comark
```

::

## Quick Start

Pick your framework and get rendering in seconds.

### Vue

::code-group

```vue [App.vue]
<script setup>
import { Comark } from 'comark/vue'

const content = `# Hello World

This is **markdown** with a custom component:

::alert{type="info"}
Welcome to Comark!
::
`
</script>

<template>
  <Comark>{{ content }}</Comark>
</template>
```

```html [Output]
<h1 id="hello-world">Hello World</h1>
<p>This is <strong>markdown</strong> with a custom component:</p>
<div class="alert alert-info">
  <p>Welcome to Comark!</p>
</div>
```

::

::callout{icon="i-lucide-triangle-alert" color="warning"}
When using async plugins (like syntax highlighting), wrap `<Comark>` in `<Suspense>`:
```vue
<Suspense>
  <Comark :plugins="[highlight()]">{{ content }}</Comark>
</Suspense>
```
::

### React

::code-group

```tsx [App.tsx]
import { Comark } from 'comark/react'

const content = `# Hello World

This is **markdown** with a custom component:

::alert{type="info"}
Welcome to Comark!
::
`

export default function App() {
  return <Comark>{content}</Comark>
}
```

```html [Output]
<h1 id="hello-world">Hello World</h1>
<p>This is <strong>markdown</strong> with a custom component:</p>
<div class="alert alert-info">
  <p>Welcome to Comark!</p>
</div>
```

::

### HTML (No Framework)

For server-side rendering or static generation without a framework:

::code-group

```typescript [render.ts]
import { parse } from 'comark'
import { renderHTML } from 'comark/string'

const tree = await parse(`# Hello World

This is **markdown** with a custom component:

::alert{type="info"}
Welcome to Comark!
::
`)

const html = renderHTML(tree)
console.log(html)
```

```html [Output]
<h1 id="hello-world">Hello World</h1>
<p>This is <strong>markdown</strong> with a custom component:</p>
<alert type="info">
  <p>Welcome to Comark!</p>
</alert>
```

::

::callout{icon="i-lucide-info" color="info"}
Use the `components` option in `renderHTML` to control how custom components are rendered to HTML. See [HTML Rendering](/rendering/html) for details.
::
