Install Comark using your preferred package manager:

::code-group

```bash [npm]
npm install comark
```

```bash [pnpm]
pnpm add comark
```

```bash [yarn]
yarn add comark
```

```bash [bun]
bun add comark
```

::

## Simple usage


```vue
<script setup lang="ts">
import { Comark } from 'comark/vue'

const content = '# Hello World\n\nThis is a **test**.'
</script>

<template>
  <Comark :markdown="content" />
</template>
```
