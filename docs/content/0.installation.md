# Installation

Get started with MDC Syntax in your project.

## Package Manager

Install MDC Syntax using your preferred package manager:

::code-group

```bash [npm]
npm install mdc-syntax
```

```bash [pnpm]
pnpm add mdc-syntax
```

```bash [yarn]
yarn add mdc-syntax
```

```bash [bun]
bun add mdc-syntax
```

::

## Requirements

- **Node.js**: Version 18 or higher
- **TypeScript**: Version 5.0 or higher (optional, but recommended)
- **Vue**: Version 3.3 or higher (only for Vue integration features)

## Basic Setup

### TypeScript Configuration

If you're using TypeScript, make sure your `tsconfig.json` includes:

```json
{
  "compilerOptions": {
    "moduleResolution": "bundler",
    "strict": true,
    "esModuleInterop": true
  }
}
```

### Import the Parser

```typescript
// Core parsing API
import { parse } from 'mdc-syntax'

// Auto-close utilities
import { autoCloseMarkdown, detectUnclosedSyntax } from 'mdc-syntax'

// Streaming API
import { parseStream, parseStreamIncremental } from 'mdc-syntax/stream'

// Vue integration (requires Vue 3)
import { MDCRenderer, useMDCStream } from 'mdc-syntax/vue'
```

## Quick Start Examples

### Parse Markdown String

```typescript
import { parse } from 'mdc-syntax'

const content = `# Hello World

This is **markdown** with MDC components.

::alert{type="info"}
Important message here
::`

const result = parse(content)
console.log(result.body) // MDC AST
```

### Parse from Stream

```typescript
import { parseStreamIncremental } from 'mdc-syntax/stream'

const response = await fetch('/content.md')

for await (const result of parseStreamIncremental(response.body!)) {
  if (!result.isComplete) {
    console.log('Streaming...', result.body.value.length, 'elements')
  }
  else {
    console.log('Complete!', result.toc)
  }
}
```

### Render in Vue

```vue
<script setup lang="ts">
import { parse } from 'mdc-syntax'
import { MDCRenderer } from 'mdc-syntax/vue'

const content = '# Hello World\n\nThis is a **test**.'
const result = parse(content)
const mdcAst = result.body
</script>

<template>
  <MDCRenderer :body="mdcAst" />
</template>
```
